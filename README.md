Project Title
Marketing Campaign Performance Analytics
Project Overview

This project analyzes digital marketing campaign performance using Python, MySQL, Excel, Power BI, and Tableau. The goal is to evaluate advertising effectiveness, campaign ROI, customer engagement, and revenue generation.

The project demonstrates a complete data analytics workflow including data cleaning, SQL analysis, dashboard creation, and business insights.
